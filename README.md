# Board
Доска объявлений
1.	Регистрация
  •	ФИО
  •	Дата рождения
  •	Телефон
  •	Имя профиля
  2.	Объявление
  •	Автомобили
    o	Марка
    o	Год выпуска
    o	Пробег
    o	Цена
    o	фото
  •	Недвижимость
    o	Тип недвижимости
    o	Адрес
    o	Цена
    o	Площадь
    o	фото
3.	Редактирование объявления
  •	Изменение всех вышестоящих пунктов
4.	Взаимодействия
  •	Возможность ответить на объявление
  •	Возможность оставить комментарий
  •	Возможность удалить объявление
5.	Поиск объявлений
  •	Поиск по категориям
  •	Поиск по названию объявления
  •	Сортировка по следующим пунктам: “цена, дата выпуска/постройки, район и тд.”
